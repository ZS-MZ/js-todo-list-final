# js-todo-list-final

پروژه نهایی دوره JavaScript Mastery - ساخت یک اپلیکیشن Todo List حرفه‌ای

## ویژگی‌های پیاده‌سازی شده

### ویژگی‌های الزامی:
- افزودن تسک با متن و تاریخ سررسید اختیاری
- مارک کردن به عنوان کامل شده (با خط خوردن و رنگ سبز)
- حذف تسک
- فیلتر کردن (همه / در حال انجام / کامل شده)
- ذخیره دائمی با localStorage (بعد از رفرش هم می‌مونه)

### ویژگی‌های بونوس اضافه شده:
- ویرایش تسک با دابل کلیک روی متن
- اضافه کردن تاریخ سررسید + هایلایت قرمز برای تسک‌های overdue
- نمایش آمار تعداد تسک‌های کامل شده
- دکمه پاک کردن همه تسک‌های کامل شده

## چیزهایی که یاد گرفتم
در این پروژه تقریباً همه مفاهیم دوره رو استفاده کردم: آرایه و آبجکت، map/filter/reduce، DOM manipulation، event listener ها، localStorage و کار با Bootstrap.  
فهمیدم که چقدر مهمه که بعد از هر تغییر داده، لیست رو دوباره رندر کنیم و localStorage رو آپدیت کنیم. همچنین ویرایش inline خیلی تجربه کاربری رو بهتر کرد.
